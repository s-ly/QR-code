# QR-code  
Тестовое задание для программиста  
## Задание  
Сгенерировать два QR-кода, положить их на столе и померить через приложение Юнити расстояние между двумя QR.
## Описание  
Версия Unity 2022.3.21f1  
Управление: AWSD + QERB + Мышка.
